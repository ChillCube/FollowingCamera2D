# FollowingCamera2D

A camera node that lets you simply pick a node and have the camera track the position of that node

## Installation

Make sure to download the ChillCube Developer tools: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Using the Godot Plugin
Enable the **ChillCube Tools** plugin, then paste the repo URL into the Clone tab.

### Linux / CLI
```bash
clone-gd-addon https://github.com/ChillCube/FollowingCamera2D.git
```

## Documentation
https://github.com/ChillCube/FollowingCamera2D/blob/main/DOCUMENTATION.md

## Deinstallation
Use the CC Tools panel in Godot, or run:
```bash
remove-gd-addon FollowingCamera2D
```

## For Developers
```bash
push-all-addons
```
